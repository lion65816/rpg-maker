★ 使用方法
--------------------------------------------------------------------------------

1.スクリプトで最初のセクションに 共通スクリプト rev-29以降
  (内容は、同梱の base.rb にあります。)を新規に追加します。
  既に入っている場合はこの操作は必要ありません。

2.Mainセクションを 共通実行スクリプト rev-29以降
  (内容は、同梱の main.rb にあります。)で上書きします。
  既に入っている場合はこの操作は必要ありません。

3. antiHangup-EX ハングアップ回避スクリプト 
  (内容は、同梱の antiHangup-EX.rb にあります。)を入れます。
  素材箇所に入れてください。

4.antiHangup.dllをゲームディレクトリに入れます。

5.デバッグをする場合はデバッグ修正スクリプトを導入します。
  (導入しませんと、デバッグ時に保護できません。)


================================================================================
★ リファレンス
--------------------------------------------------------------------------------
  AntiHangup モジュール
--------------------------------------------------------------------------------
  クラスメソッド
--------------------------------------------------------------------------------
  start
--------------------------------------------------------------------------------
  ハングアップ保護を開始します。
この間、処理速度が低下します。

※一つのメソッドが10秒以上待たされるものは回避できません。
  例えば、DLLの呼び出しで10秒以上掛かるものです。


--------------------------------------------------------------------------------
  stop
--------------------------------------------------------------------------------
  ハングアップ保護を終了します。
